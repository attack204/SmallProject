## 常见的文本标签

首先，所有的标签都会与生俱来的带有很多边距，如果想要取消的话

```html
    * {
        margin-top: 0px;
    }
```

1.换行标签


`<nobr></nobr>` 不换行，比如很多图片，可以用nobr强制不换行

`<br>` 换行

`<hr>` 换行+水平线

`<p></p>` 段落

关于这个p标签，有一个自己独特的属性就是对其方式

`<center></center>` 大面积居中

`<pre></pre>`以原格式输出

```html
    <ul>
        <li> </li>
        <li> </li>
    </ul>
    ul是无序列表
    ol是有序列表
    <dl>
        <dt></dt>
        <dd></dd>
    </dl>
    自定义标签
```
